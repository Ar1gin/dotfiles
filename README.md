## Installation
Dotfiles are managed by `stow` via `make`.

Apply dotfiles
```bash
make stow
```
Unapply dotfiles
```bash
make unstow
```
Reapply dotfiles
```bash
make restow
```
These commands are here mostly as a reminder, unless you want to use all of my dotfiles.
